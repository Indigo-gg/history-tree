# 初中历史知识树 App 信息架构与原型

## 1. 产品定位

面向河南省濮阳市中考历史复习，服务对象是人教版七、八、九年级历史学习者。产品重点不是“刷大量题”，而是帮助学生把时间、事件、人物、制度、影响和易混点串成结构化知识网络。

核心目标：

- 把零散知识点整理成可展开的知识树。
- 用时间轴解决时间和事件记忆混乱。
- 用对比卡解决易混知识点。
- 用搜索和复习包适配开卷考试与考前整理。
- 用 AI 辅助解释、生成记忆法和生成小测，但不让 AI 直接决定标准知识答案。

## 2. 第一版范围

第一版做成离线优先的网页应用，后续通过 Capacitor 打包成 Android App。

技术路线：

- 前端：React + Vite + TypeScript。
- UI：CSS Modules 或 Tailwind CSS，优先做轻量自定义组件。
- 图形知识树：React Flow。
- 本地数据：JSON 知识库。
- 学习进度：IndexedDB。
- 离线能力：PWA。
- App 打包：Capacitor。
- AI 接入：前端配置个人 API Key，或后续接 Cloudflare Worker 转发。

第一版功能：

- 知识树
- 时间轴
- 易混点
- 搜索索引
- 考前复习包
- 简易小测
- AI 辅助解释

暂不做：

- 用户账号系统
- 多人同步
- 排行榜
- 复杂主观题自动判分
- 整本教材 PDF 内置

## 7. 简易小测设计

第一版小测以自动生成的客观题为主。

题型：

- 时间排序题
- 事件匹配题
- 判断题
- 关键词填空
- 易混点选择题

生成方式：

- 从 `startYear`、`keywords`、`examPhrases`、`confusionIds` 生成题干。
- 干扰项优先从同专题、相近年份、易混节点中取。
- AI 可以辅助生成解释，但题目答案由结构化数据决定。

示例：

```text
题目：下列哪一项最能说明洋务运动的历史地位？
A. 推翻了清朝统治
B. 是中国近代化的开端
C. 建立了资产阶级共和国
D. 促进了马克思主义传播

答案：B
解释：洋务运动学习西方先进技术，创办近代工业，是中国近代化的开端。
```

## 8. AI 接入边界

AI 适合做：

- 把知识点讲得更口语化。
- 生成记忆口诀。
- 解释易混原因。
- 根据当前节点生成 3-5 道练习。
- 把一组知识点总结成复习短文。

AI 不适合直接做：

- 标准答案来源。
- 中考政策判断。
- 教材原文替代。
- 无审核地批量生成知识库。

建议 AI Prompt：

```text
你是河南中考历史复习助手。请只依据下面给出的结构化知识点解释，不要补充未经提供的史实。
目标学生是初三学生，解释要清楚、短、适合考前复习。
请输出：一句话理解、容易混淆点、记忆提示。
```

## 9. 视觉风格

整体风格要像学习工具，不像营销网站。

建议：

- 背景：浅灰白，减少视觉疲劳。
- 主色：沉稳蓝或青绿，用于当前节点和主操作。
- 辅色：黄色表示易混，红色表示错点，绿色表示掌握。
- 字体：系统默认中文字体。
- 卡片圆角：6-8px。
- 知识树节点保持紧凑，避免大卡片堆叠。
- 不使用大面积渐变和装饰图形。

桌面端重点是信息密度；移动端重点是清晰分层。

### 9.1 轻像素风方向

可以加入轻量像素风，但它只作为识别感和情绪缓冲，不影响阅读效率。

建议做法：

- 节点图标使用 16px 或 24px 像素风小图标，例如卷轴、书本、城墙、火炬、指南针、齿轮。
- 页面空状态、复习完成状态使用像素风插图。
- 知识树节点仍保持现代卡片样式，只在左侧加像素图标或像素角标。
- 背景可以使用极淡的网格纹理，透明度控制在 3%-5%。
- 避免全站像素字体。正文必须使用清晰的系统中文字体。
- 像素元素的边缘清晰，不做模糊阴影。

建议素材尺寸：

- 图标：32x32 PNG，透明背景。
- 插图：512x512 PNG，透明背景。
- 首页横幅：1024x360 PNG，低饱和背景，不放大段文字。
